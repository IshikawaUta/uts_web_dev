# **App Name**: LuminaFolio

## Core Features:

- Project Showcase: Visually appealing display of projects with descriptions, images, and links to live sites or repositories.
- About Me Section: A personal introduction, skills overview, and contact information.
- Interactive Resume: Display an interactive version of your resume.
- Contact Form: Allow visitors to send direct messages.
- AI-Powered Bio Generator: An AI tool suggests compelling profile bios based on keywords and experience.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and modern feel.
- Background color: Very light grayish-blue (#F0F4F9).
- Accent color: Electric Violet (#8F72D1) for highlights and interactive elements.
- Headline font: 'Space Grotesk' sans-serif for a computerized, techy feel.
- Body font: 'Inter' sans-serif for readability and a clean aesthetic.
- Use minimalist, line-based icons for navigation and to represent skills. Ensure consistency in style and weight.
- Implement a clean, grid-based layout with plenty of white space to ensure readability and focus on showcased work. Use clear visual hierarchy.
- Subtle transitions and hover effects to create a polished and engaging user experience without being distracting.
